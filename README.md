# Calculadora 

Calculadora Java - Programação Orientada a Objetos

Este é um projeto de calculadora desenvolvido em JAVA com o objetivo de aplicar fundamentos de Programação Orientada a Objetos (POO), controle de versão com Git e testes automatizados.

# Funcionalidades
- Operações básicas: Soma, Subtração, Multiplicação e Divisão.
- Entrada de dados interativa via console (Scanner).
- Tratamento de exceções (evita erros ao digitar letras ou dividir por zero).
- Testes Unitários: Garantia de funcionamento via JUnit.

# Tecnologias e Conceitos Aplicados
- Linguagem: Java 17+
- Interface: Uso de contrato para padronizar o método de cálculo.
- Encapsulamento: Atributos privados com métodos Getters e Setters.
- JUnit 5: Implementação de testes automatizados para validar a lógica.
- Git/GitHub: Controle de versão e histórico de commits.

# Como Rodar os Testes
Para garantir que a calculadora está funcionando corretamente, execute a classe `CalculadoraTest.java` como JUNIT TESTE no seu ambiente de desenvolvimento.

# strutura do Projeto
- `Calculadora.java`: Contém a lógica de negócio e atributos.
- `Calculo.java`: Classe principal para interação com o usuário.
- `interCalculadora.java`: Interface que define o contrato do sistema.
- `CalculadoraTest.java`: Conjunto de testes automatizados.
